# Parking System - Fullstack Starter

## Overview
This project is a starter fullstack parking reservation system that matches the ERD you provided.
Backend: Node.js + Express + MySQL
Frontend: React (create-react-app style)

## Steps to run

1. Install MySQL and run the schema:
   mysql -u root -p < backend/schema.sql

2. Backend:
   cd backend
   copy .env.example to .env and edit DB_PASS if needed
   npm install
   npm start

3. Frontend:
   cd frontend
   npm install
   npm start

API default: http://localhost:3001
